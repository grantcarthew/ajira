# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.4.0] - 2026-01-28

### Changed

- Default issue view now shows 5 comments instead of 3
- Updated project attachment support status documentation

### Fixed

- Code formatting issues in attachment list and issue view commands
- Test environment variable cleanup for config tests

## [0.3.0] - 2026-01-28

### Added

- Issue attachment commands (add, list, download, remove)
- File size formatting for attachment display

## [0.2.0] - 2026-01-16

### Added

- Sprint management commands
- Board management commands
- Epic management commands

## [0.1.0] - 2026-01-09

### Added

- Issue management commands (create, view, list, edit, clone, move)
- Comment functionality
- Project and field commands
- Markdown to ADF conversion
- JSON output support

## [0.0.1] - 2026-01-05

### Added

- Initial release
- Basic authentication with Jira Cloud
- User verification (me command)
